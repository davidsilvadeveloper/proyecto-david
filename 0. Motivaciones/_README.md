## En qué orden leer esto

1. `_bienvenida.md` — el primer archivo. 2 minutos. Léelo hoy.
2. `_por_que_el_icfes_importa.md` — el argumento honesto, sin discurso
3. `_como_usar_este_sistema.md` — cómo moverte por Tareitas
4. `_miKit_de_Prompts.md` — guárdalo. Lo vas a necesitar.
