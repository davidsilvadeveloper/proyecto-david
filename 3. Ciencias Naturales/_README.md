# 🔬 Ciencias Naturales — Guía de navegación

> El ICFES no te pide memorizar. Te pide pensar como científica. Ya puedes hacerlo.

## Por dónde empezar

1. Lee `_listaTematica.md` — los 3 bloques y la clave secreta
2. Empieza por el bloque del tema que más te llame (Biología, Química o Física)
3. Siempre abre PhET después de leer el contenido — la simulación consolida lo que leíste

| Carpeta | Qué entrena | Recurso estrella |

| 3.1 Conocimiento Cientifico | Célula, átomo, cuerpo humano | PhET + Amoeba Sisters |
| 3.2 Por que pasan las cosas | Newton, energía, ecosistemas | PhET + Kurzgesagt |
