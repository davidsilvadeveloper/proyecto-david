# ESTRUCTURA UNIVERSAL

```text

```

# REPRESENTACIONES EQUIVALENTES

| Entrada | Proceso | Salida |
|---|---|---|
| Problema | Procedimiento | Solución |
| Pregunta | Pasos | Respuesta |

- estructurada,
- verificable,
- orientada a resolver problemas.

# 3. MODELO UNIVERSAL DE COMPRENSIÓN

## ELEMENTOS

Son las partes fundamentales de un sistema.

---

## RELACIONES

Son las conexiones entre elementos.

---

## REGLAS

Son las leyes que gobiernan las relaciones.

# DEFINICIÓN OPERATIVA DE ENTENDER

Comprender algo significa:

1. Identificar sus elementos.
2. Comprender sus relaciones.
3. Descubrir sus reglas.
4. Poder representarlo.
5. Poder explicarlo.
6. Poder aplicarlo.
